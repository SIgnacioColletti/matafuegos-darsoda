## üìñ Descripcion

Sitio web moderno y responsive desarrollado para Matafuegos Dar Soda SRL, una empresa familiar con mas de 25 anos de trayectoria en el rubro de seguridad contra incendios. El sitio incluye catalogo de productos, informacion de servicios, formulario de contacto integrado con WhatsApp y optimizacion SEO local.

### Objetivos del Proyecto

- Presencia digital profesional para la empresa
- Catalogo de productos con consulta directa por WhatsApp
- Generacion de leads a traves de formulario de contacto
- Posicionamiento SEO local en Rosario y Santa Fe
- Experiencia de usuario optimizada en dispositivos moviles
